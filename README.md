# EssaIDE

EssaIDE is an AI-powered, terminal-based Integrated Development Environment (IDE) built entirely in Python. It is designed to natively integrate with the `Essa` suite of enterprise microservices:

- **EssaSearch:** High-speed, global codebase searching
- **EssaCache:** In-memory AST and symbol caching
- **EssaConnect:** Real-time event bus
- **EssaDB:** Persistent workspace storage
- **EssaProxy:** API Gateway

## Getting Started

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the IDE:
```bash
python main.py
```
